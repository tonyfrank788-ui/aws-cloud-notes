# Lesson 2 - Types of Cloud Computing

## Public Cloud
Resources are owned by a cloud provider like AWS and shared among many customers.

## Private Cloud
Resources are dedicated to a single organization for greater control and security.

## Hybrid Cloud
A combination of public and private clouds, allowing organizations to use both.

### Easy way to remember

- Public Cloud = Hotel 🏨
- Private Cloud = Your House 🏠
- Hybrid Cloud = House + Hotel 🏠🏨
