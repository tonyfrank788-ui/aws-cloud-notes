# Lesson 3 - AWS Regions

## What is an AWS Region?
An AWS Region is a geographic location where AWS has data centers.

## Four factors when choosing a Region
1. Latency - Choose a Region close to your users.
2. Price - Costs vary by Region.
3. Service Availability - Not every service is available in every Region.
4. Compliance - Follow local laws and regulations.

## Key Point
AWS Regions are independent, and data is not automatically copied between Regions.
